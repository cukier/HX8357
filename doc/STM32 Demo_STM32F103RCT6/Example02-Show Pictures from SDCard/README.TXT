测试开发板：
正点原子ALIENTEK MiniSTM32 开发板
单片机型号：STM32F103RCT6
本实验将实现如下功能：开机的时候先检测SD卡是否存在，然后初始化FAT文件系统，在这之后开始查找根目录下的PICTURE文件夹，如果找到则显示该文件夹下面的图片，循环显示，通过按KEY0和KEY1可以快速浏览下一张和上一张。如果未找到图片文件夹/图片，则提示错误。同样我们也是用LED0来指示程序正在运行。

注意:
1,本实验可以SWD下载,但是不能仿真(得不到正确的结果).所以下载完了,请拔掉JLINK.
2,不要把1820和PA0的跳线帽跳上,否则可能导致按键"不灵".
3,本实验FATFS不支持中文文件名，所以中文名的图片无法显示！
